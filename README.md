# Tao Su's Blog

My personal blog and portfolio built with Astro, Tailwind CSS, and TypeScript.

## About

This is where I share my projects, lessons learned, and thoughts on tech and learning. The site is statically generated for fast performance and optimized for SEO.

## Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** TypeScript
- **Interactivity:** [Solid.js](https://www.solidjs.com/)
- **Package Manager:** [Bun](https://bun.sh/)
- **Deployment:** [Vercel](https://vercel.com/)

## Features

- 100/100 Lighthouse performance
- Responsive and accessible design
- Light/Dark theme with smooth transitions
- Full-text search across posts and projects
- JSON-LD structured data for SEO
- Link prefetching on hover
- Auto-generated sitemap and RSS feed
- Markdown and MDX support
- Code blocks with copy-to-clipboard

## Commands

All commands are run from the root of the project:

| Command          | Action                                     |
| :--------------- | :----------------------------------------- |
| `bun install`    | Install dependencies                       |
| `bun run dev`    | Start local dev server at `localhost:4321` |
| `bun run build`  | Build production site to `./dist/`         |
| `vercel dev`     | Preview build locally with Vercel CLI      |
| `bun run format` | Format code with Prettier                  |
| `bun run lint`   | Run ESLint                                 |
